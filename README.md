# dotvim

my personal Vim config (with pathogen and plugins as git submodules)


## Installation

```
$ git clone https://github.com/javierav/dotvim.git .vim
$ cd .vim
$ git submodule update --init
```


## Update plugins

```
$ ./update-plugins.sh
```
